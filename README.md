# Math Quiz (C++)

A console-based math quiz game built in C++. The player selects a difficulty level and operation type, then answers a series of randomly generated arithmetic questions, with immediate feedback and a final score summary.

## Features

- Configurable number of questions per session
- Difficulty levels: Easy, Medium, Hard, or Mixed (controls the range of numbers used)
- Operation types: Addition, Subtraction, Multiplication, Division, or Mixed
- Randomly generated questions based on selected level and operation
- Color-coded console feedback (correct/incorrect) with sound alert on wrong answers
- Final results summary: number of correct/incorrect answers and overall pass/fail outcome
- Option to replay without restarting the program

## How to Run

1. Open `Course5_Project2.vcxproj` in Visual Studio
2. Build and run the project (F5)
3. Follow the on-screen prompts to play

## Technologies

- C++
- Standard I/O (`iostream`)
